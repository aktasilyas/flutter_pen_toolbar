# Assets

This folder contains screenshots and preview images for the package.

## Required Screenshots

For pub.dev listing, add the following screenshots:

1. `screenshot_1.png` - Main toolbar with various pen types
2. `screenshot_2.png` - Pen selection animation
3. `preview.gif` - Animated preview (optional)

## Screenshot Guidelines

- Recommended size: 1080x1920 or 1920x1080
- Format: PNG for static, GIF for animated
- Show the toolbar with different pen selections
- Use a clean background
